         - Context Variables:

         - Renders inquiry.html
         - Context Variables:

           - errors (dict, optional): Validation or submission errors

         - On validation failure, re-renders inquiry.html with errors and properties context
         - On success, redirects to inquiries list page

         - Renders inquiries.html
         - Context Variables:
           - inquiries (list of dict): All user inquiries

         - Context Variables:

         - Redirects to favorites list page
