# Flask Routes and Context Variables

## Routes

1. Route: /
   - Function: redirect_dashboard
   - Methods: GET
   - Purpose: Redirects to /dashboard

2. Route: /dashboard
   - Function: dashboard_page
   - Methods: GET
   - Context Variables:
     - featured_restaurants (List[Dict]): List of featured restaurants with keys: id, name, type, address, phone, rating, delivery_time, delivery_fee

3. Route: /restaurants
   - Function: restaurants_page
   - Methods: GET
   - Context Variables:
     - restaurant_list (List[Dict]): List of restaurants with keys: id, name, type, address, phone, rating, delivery_time, delivery_fee
     - filter_type (Optional[str]): Type of cuisine to filter restaurants

4. Route: /menu/<int:restaurant_id>
   - Function: restaurant_menu_page
   - Methods: GET
   - Context Variables:
     - restaurant_info (Dict): Details of the restaurant (id, name, type, address, phone, rating, delivery_time, delivery_fee)
     - menu_items (List[Dict]): List of menu items with keys: item_id, name, category, description, price, image_url (optional)

5. Route: /item/<int:item_id>
   - Function: item_details_page
   - Methods: GET
   - Context Variables:
     - item_info (Dict): Information about the item (id, name, description, price, image_url (optional))

6. Route: /cart
   - Function: shopping_cart_page
   - Methods: GET
   - Context Variables:
     - cart_items (List[Dict]): List of items in cart with keys: item_id, name, quantity, price
     - total_amount (float): Total price of items in cart

7. Route: /checkout
   - Function: checkout_page
   - Methods: GET, POST
   - Context Variables:
     - delivery_address (Optional[str]): Address entered by user (on GET shows empty or saved)

8. Route: /orders
   - Function: orders_page
   - Methods: GET
   - Context Variables:
     - active_orders (List[Dict]): List of active orders with keys: order_id, restaurant_name, status, delivery_address
     - filter_status (str): Current filter status (All, Preparing, Delivered)

9. Route: /tracking/<int:order_id>
   - Function: tracking_page
   - Methods: GET
   - Context Variables:
     - delivery_details (Dict): Delivery person name, phone, vehicle, status
     - timeline_updates (List[Dict]): List of status updates with timestamps

10. Route: /reviews/<int:restaurant_id>
    - Function: reviews_page
    - Methods: GET, POST
    - Context Variables:
      - reviews_list (List[Dict]): List of reviews with keys: review_id, customer_name, rating, review_text, review_date
      - restaurant_name (str): Name of the restaurant

---

## Note:
- Dynamic segments represent variables parsed from URLs, e.g., <int:restaurant_id>.
- All functions implement GET methods except checkout and reviews which handle POST where applicable.
- These variables allow template rendering for corresponding pages.
