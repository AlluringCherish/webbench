# Data Schema for NewsPortal

# Articles Data Schema
# Columns: article_id|category|title|author|summary|publish_date|popularity_score|content
# Description: Stores article details
# Example:
# 1|Technology|Breakthrough in Computing|John Doe|A new computing breakthrough...|2025-01-20|100|Full article content here

# Categories Data Schema
# Columns: category_id|category_name|description
# Description: Stores categories information
# Example:
# 1|Technology|Latest technology news and events

# Bookmarks Data Schema
# Columns: bookmark_id|user_id|article_id|date_saved
# Description: Stores user bookmarks
# Example:
# 1|1|1|2025-01-20

# Comments Data Schema
# Columns: comment_id|article_id|commenter_name|comment_text|comment_date
# Description: Stores comments on articles
# Example:
# 1|1|Alice|Great article!|2025-01-21

# Trending Articles Schema
# Columns: rank|category|title|popularity|time_period
# Description: Tracks trending articles by rank
# Example:
# 1|Sports|Championship Victory|3210|This Week
