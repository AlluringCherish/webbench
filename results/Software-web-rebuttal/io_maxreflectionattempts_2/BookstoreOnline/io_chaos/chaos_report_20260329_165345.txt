         ### 1. Books Data

           - `isbn` (str)
           - `category` (str)
           - `price` (float)
           - `stock` (int)
           - `description` (str)
         - **Description:** Stores all books with identifying info, categorization, price, stock count, and description
         - **Example Rows:**
           ```
           1|The Great Gatsby|F. Scott Fitzgerald|9780743273565|Fiction|12.99|50|A classic American novel
           2|Sapiens|Yuval Noah Harari|9780062316097|Non-Fiction|16.99|30|A brief history of humankind

           ```

           - `description` (str)
         - **Description:** Stores all categories available for classifying books
         - **Example Rows:**
           ```
           1|Fiction|Fictional narratives and novels
           2|Non-Fiction|Factual and educational books
           3|Science|Scientific topics and research
           ```

           ```
           1|1|2|2025-01-15
           2|3|1|2025-01-16
           ```

           - `order_date` (str: YYYY-MM-DD)
           - `total_amount` (float)
           - `status` (str) (e.g., Pending, Shipped, Delivered)
           - `shipping_address` (str)

         - **Example Rows:**

         - **Description:** Redirects to `/dashboard`.
         
         ---
         
         ### 2. Dashboard Page
         - **Route Path:** `/dashboard`
         - **Function Name:** `dashboard_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `dashboard.html`
         - **Context Variables:**
           - `featured_books` (list of dict): Each dict containing `book_id` (int), `title` (str), `author` (str), `price` (float)
           - `bestsellers` (list of dict): Each dict containing `book_id` (int), `title` (str), `author` (str), `sales_count` (int)
         
         - **Description:** Displays featured books and bestsellers with navigation buttons.
         
         ---
         
         ### 3. Book Catalog Page
         - **Route Path:** `/catalog`
         - **Function Name:** `book_catalog_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `catalog.html`
         - **Context Variables:**
           - `books` (list of dict): Each dict with `book_id` (int), `title` (str), `author` (str), `isbn` (str), `category` (str), `price` (float)
           - `categories` (list of dict): Each dict with `category_id` (int), `category_name` (str)
           - `search_query` (str): Current search filter
           - `selected_category` (str): Current category filter
         
         - **Description:** Enables book search by title, author, or ISBN and filtering by category.
         
         ---
         
         ### 4. Book Details Page
         - **Route Path:** `/book/<int:book_id>`
         - **Function Name:** `book_details_page`
         - **HTTP Methods:** GET, POST
         - **Template Rendered:** `book_details.html`
         - **Context Variables:**
           - `book` (dict): Fields `book_id` (int), `title` (str), `author` (str), `isbn` (str), `category` (str), `price` (float), `stock` (int), `description` (str)
           - `reviews` (list of dict): Each dict has `review_id` (int), `customer_name` (str), `rating` (int), `review_text` (str), `review_date` (str)
         
         - **Form Submission (POST):** Add selected quantity of the book to cart.
           - Input: Quantity (int)
           - Actions: Add or update item in `cart.txt`.
           - Redirect: Back to same details page.
         
         ---
         
         ### 5. Shopping Cart Page
         - **Route Path:** `/cart`
         - **Function Name:** `shopping_cart_page`
         - **HTTP Methods:** GET, POST
         - **Template Rendered:** `cart.html`
         - **Context Variables:**
           - `cart_items` (list of dict): Each dict with `cart_id` (int), `book_id` (int), `title` (str), `quantity` (int), `price` (float), `subtotal` (float)
           - `total_amount` (float)
         
         - **Form Submissions (POST):**
           - Update quantity of cart item (`cart_id` and `quantity` in form)
           - Remove item from cart (`cart_id` in form)
         
         - **Redirects:** Back to `/cart` after update
         - **Buttons:** Proceed to checkout navigates to `/checkout`
         
         ---
         
         ### 6. Checkout Page
         - **Route Path:** `/checkout`
         - **Function Name:** `checkout_page`
         - **HTTP Methods:** GET, POST
         - **Template Rendered:** `checkout.html`
         - **Context Variables:**
           - `cart_items` (list of dict): Same structure as shopping cart
           - `total_amount` (float)
         
         - **Form Submission (POST):** Place order
           - Inputs: `customer_name` (str), `shipping_address` (str), `payment_method` (str)
           - Actions: Write new order to `orders.txt` and items to `order_items.txt` and clear cart
           - Redirect: `/order_history`
         
         ---
         
         ### 7. Order History Page
         - **Route Path:** `/order_history`
         - **Function Name:** `order_history_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `orders.html`
         - **Context Variables:**
           - `orders` (list of dict): Fields `order_id` (int), `customer_name` (str), `order_date` (str), `total_amount` (float), `status` (str), `shipping_address` (str)
           - `selected_status` (str): Current status filter ("All" or specific)
         
         - **Functionality:** Filter orders by status via query parameter
         
         ---
         
         ### 8. Order Details Page
         - **Route Path:** `/order/<int:order_id>`
         - **Function Name:** `order_details_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `order_details.html`
         - **Context Variables:**
           - `order` (dict): Fields `order_id`, `customer_name`, `order_date`, `total_amount`, `status`, `shipping_address`
           - `order_items` (list of dict): Each has `order_item_id`, `book_id`, `title`, `quantity`, `price`
         
         ---
         
         ### 9. Reviews Page
         - **Route Path:** `/reviews`
         - **Function Name:** `reviews_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `reviews.html`
         - **Context Variables:**
           - `reviews` (list of dict): Each dict has `review_id` (int), `book_id` (int), `title` (str), `customer_name` (str), `rating` (int), `review_text` (str), `review_date` (str)
           - `filter_rating` (int or None): Current rating filter
         
         - **Functionality:** Filter reviews by rating
         - **Navigation:** Button to `/write_review`
         
         ---
         
         ### 10. Write Review Page
         - **Route Path:** `/write_review`
         - **Function Name:** `write_review_page`
         - **HTTP Methods:** GET, POST
         - **Template Rendered:** `write_review.html`
         - **Context Variables:**
           - `purchased_books` (list of dict): Each dict with `book_id` (int), `title` (str)
         
         - **Form Submission (POST):** Submit review
           - Inputs: `book_id` (int), `rating` (int), `review_text` (str), `customer_name` (str optional or fixed)
           - Actions: Append new review line to `reviews.txt` with current date
           - Redirect: `/reviews`
         
         ---
         
         ### 11. Bestsellers Page
         - **Route Path:** `/bestsellers`
         - **Function Name:** `bestsellers_page`
         - **HTTP Methods:** GET
         - **Template Rendered:** `bestsellers.html`
         - **Context Variables:**
           - `bestsellers` (list of dict): Each dict with `book_id` (int), `title` (str), `author` (str), `sales_count` (int), `period` (str)
           - `time_period` (str): Current time period filter ("This Week", "This Month", "All Time")
         
         - **Functionality:** Filter bestsellers by time period
         - **Navigation:** Buttons to book details and back to dashboard

         ## Section 3: Data Schemas Specification
         
         ### 1. Books Data
         - **File Path:** `data/books.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `book_id` (int)
           2. `title` (str)
           3. `author` (str)
           4. `isbn` (str)
           5. `category` (str)
           6. `price` (float)
           7. `stock` (int)
           8. `description` (str)
         - **Description:** Stores books including descriptive and stock information
         - **Example Rows:**
           ```
           1|The Great Gatsby|F. Scott Fitzgerald|9780743273565|Fiction|12.99|50|A classic American novel
           2|Sapiens|Yuval Noah Harari|9780062316097|Non-Fiction|16.99|30|A brief history of humankind
           3|1984|George Orwell|9780451524935|Fiction|14.99|45|Dystopian social science fiction
           ```
         
         ---
         
         ### 2. Categories Data
         - **File Path:** `data/categories.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `category_id` (int)
           2. `category_name` (str)
           3. `description` (str)
         - **Description:** Stores book categories
         - **Example Rows:**
           ```
           1|Fiction|Fictional narratives and novels
           2|Non-Fiction|Factual and educational books
           3|Science|Scientific topics and research
           ```
         
         ---
         
         ### 3. Cart Data
         - **File Path:** `data/cart.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `cart_id` (int)
           2. `book_id` (int)
           3. `quantity` (int)
           4. `added_date` (str in `YYYY-MM-DD` format)
         - **Description:** Tracks current shopping cart contents
         - **Example Rows:**
           ```
           1|1|2|2025-01-15
           2|3|1|2025-01-16
           ```
         
         ---
         
         ### 4. Orders Data
         - **File Path:** `data/orders.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `order_id` (int)
           2. `customer_name` (str)
           3. `order_date` (str in `YYYY-MM-DD` format)
           4. `total_amount` (float)
           5. `status` (str) (e.g. Pending, Shipped, Delivered)
           6. `shipping_address` (str)
         - **Description:** Stores orders with associated customer and shipping details
         - **Example Rows:**
           ```
           1|John Doe|2025-01-10|38.97|Delivered|123 Main St, New York, NY 10001
           2|Jane Smith|2025-01-14|16.99|Shipped|456 Oak Ave, Los Angeles, CA 90001
           ```
         
         ---
         
         ### 5. Order Items Data
         - **File Path:** `data/order_items.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `order_item_id` (int)
           2. `order_id` (int)
           3. `book_id` (int)
           4. `quantity` (int)
           5. `price` (float)
         - **Description:** Stores ordered book items
         - **Example Rows:**
           ```
           1|1|1|2|12.99
           2|1|3|1|14.99
           3|2|2|1|16.99
           ```
         
         ---
         
         ### 6. Reviews Data
         - **File Path:** `data/reviews.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `review_id` (int)
           2. `book_id` (int)
           3. `customer_name` (str)
           4. `rating` (int) (1-5)
           5. `review_text` (str)
           6. `review_date` (str in `YYYY-MM-DD` format)
         - **Description:** Stores customer reviews and ratings
         - **Example Rows:**
           ```
           1|1|Alice Johnson|5|Amazing book! A true classic.|2025-01-12
           2|2|Bob Williams|4|Very informative and well-written.|2025-01-13
           3|3|Charlie Brown|5|Thought-provoking and relevant today.|2025-01-15
           ```
         
         ---
         
         ### 7. Bestsellers Data
         - **File Path:** `data/bestsellers.txt`
         - **Format:** Pipe-delimited (`|`)
         - **Fields (in order):**
           1. `book_id` (int)
           2. `sales_count` (int)
           3. `period` (str) (e.g. "This Week", "This Month", "All Time")
         - **Description:** Sales ranking data for books
         - **Example Rows:**
           ```
           2|150|This Month
           1|120|This Month
           3|95|This Month
           ```
